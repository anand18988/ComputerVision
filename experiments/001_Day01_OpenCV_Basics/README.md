## Important Concepts

An image is simply a NumPy array.

Color images are stored using the BGR format in OpenCV.

Grayscale images contain only one channel.

## Functions Learned

cv2.imread()

Reads an image from disk.

cv2.imshow()

Displays an image.

cv2.cvtColor()

Converts between color spaces.

cv2.resize()

Changes image dimensions.

cv2.rotate()

Rotates an image.

cv2.rectangle()

Draws a rectangle.

cv2.circle()

Draws a circle.

cv2.putText()

Adds text.

cv2.imwrite()

Saves an image.
